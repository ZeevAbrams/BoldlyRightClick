Boldly Right Click

Use this extension to format text in any input field:

  1. Select text with your mouse
  2. Right-click on the selection
  3. Choose "Boldly Right Click" from the menu
  4. Select Bold, Italics, or Bold+Italics

Tip: To convert formatted text back to plain text, select it and use the "Convert to Plain Text" option.

Perfect for LinkedIn, Twitter, and other platforms without formatting options!
